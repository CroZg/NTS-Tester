# NTS-Tester
Tool for quick testing of NTS (Network Time Security) servers.<br><br>
Windows(tm) 10+ x64 command-line + GUI tool for testing NTP servers that implement the KE (Key Establishment) protocol of the Network Time Security (NTS) authentication mechanism.
# Requirements
Any Windows x64 platform (10+, 2019+)
